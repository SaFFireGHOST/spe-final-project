# Jenkins Pipeline Errors and Solutions

## 1. Gateway Pod Error (ModuleNotFoundError)
**Issue:** The `gateway` pod failed to start with `ModuleNotFoundError: No module named 'flask'`.
**Context:** Deployment stage.
**Solution:** Updated the Dockerfile/requirements to ensure `flask` is installed in the gateway service image.

## 2. ELK Stack Port Conflict
**Issue:** Kibana failed to bind to port 5601 because it was already in use.
**Context:** Local testing/Deployment.
**Solution:** Used `kubectl port-forward` to map Kibana to a different local port or identified and stopped the conflicting process.

## 3. Minikube/Kubectl Missing in Ansible
**Issue:** Ansible playbook failed or would fail if Minikube/kubectl were not ready.
**Context:** Pre-deployment checks.
**Solution:** Added `kubectl version` and `minikube status` checks to `ansible/deploy.yml` to fail early if environment is not ready.
